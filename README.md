app.py - Главный модуль приложения

- Содержит графический пользовательский интерфейс (GUI)

- Классы для кнопок, выпадающих меню

- Основная логика работы приложения

generator.py - Модуль генерации тайлов

- Алгоритмы процедурной генерации

- Работа с изображениями (Pillow)

- Преобразование спрайтов в тайлы

- Применение цветовых палитр

saved.py - Модуль сохранения

- Диалоговые окна сохранения файлов

- Поддержка разных форматов (PNG, JPEG, GIF, BMP)

- Управление путями и именами файлов

sprites3.png - Спрайт-лист

- Исходные текстуры для генерации тайлов

- Содержит 11×9 спрайтов

tests.py - Модуль тестирования

Тесты для проверки функционала

- Проверка обработки ошибок
